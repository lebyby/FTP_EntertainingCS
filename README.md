Цифровой образовательный ресурс "Занимательная информатика"
===
Full-stack веб-приложение основанное на MERN stack
***
**Реализовано:** 
1) Регистрация  
2) Два типа интерактивных задач (с помощью PixiJS)
3) Запись и просмотр результатов решения задач
